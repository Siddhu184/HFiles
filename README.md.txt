# HFiles – Medical File Management System

HFiles is a full-stack application for uploading and managing medical documents securely.

## Tech Stack

Frontend: Next.js  
Backend: .NET Web API  
Database: MySQL  

## Project Structure

HFiles
│
├── backend
│   └── HFilesAPI
│
└── frontend
    └── Next.js App

## How to Run the Project

### Backend (.NET)

1. Go to backend folder

cd backend/HFilesAPI

2. Run the API

dotnet run


### Frontend (Next.js)

1. Go to frontend folder

cd frontend

2. Install dependencies

npm install

3. Run frontend

npm run dev


Frontend runs on:
http://localhost:3000

Backend runs on:
http://localhost:7163

## Features

- User Authentication
- Upload Medical Files
- View Uploaded Files
- Secure API with JWT
